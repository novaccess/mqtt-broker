# MQTT broker docker image

## Requirements

| Dependency | Version | Description |
| ---------- | ------- | ----------- |
| Docker | >= 1.12.3 | Docker Engine |
| Docker Compose | >= 1.8.1 | Run multi docker applications|

## Pull the container

```
docker-compose pull
```

## Run the container

```
docker-compose up
```

## Build the container

```
docker-compose build
```
